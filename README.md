# 楽天レビュー AI分析システム

楽天市場のレビューをAIで分析し、マーケティングインサイトを抽出するシステムです。

## 機能

- 📊 レビューCSVをドラッグ＆ドロップで簡単分析
- 🤖 AIによる深層分析（強み・懸念点・顧客インサイト）
- 📈 改善提案とマーケティング活用アドバイス
- 📄 分析結果のPDF出力

## アクセス

https://reviewanalysis.ginzasugiden.com

## 技術構成

- **フロントエンド**: GitHub Pages (HTML/CSS/JavaScript)
- **バックエンド**: Google Apps Script (API)
- **AI**: Claude API (Anthropic)

## 利用制限

- 1店舗あたり1日10回まで
- 1回最大1000件のレビュー処理
